ContactHub Commands

  • add: Adds a person to the address book.
    Format: add n/NAME p/PHONE e/EMAIL a/ADDRESS [t/TAG]...
  • edit: Edits the details of the person identified by the index number used in the last person listing. Existing values will be overwritten by the input values.
    Format: edit INDEX [n/NAME] [p/PHONE] [e/EMAIL] [a/ADDRESS] [t/TAG]...
  • find: Finds persons whose names contain any of the given keywords.
    Format: find KEYWORD [MORE_KEYWORDS]...
  • delete: Deletes the specified person from the address book.
    Format: delete INDEX
  • select: Selects the person identified by the index number used in the last person listing.
    Format: select INDEX
  • help: Opens the User Guide
  • list: Shows a list of all persons in the address book.
  • clear: Clears all entries from the address book.
  • undo: Restores the address book to the state before the previous undoable command was executed.
  • redo: Reverses the most recent undo command.
  • exit: Exits the program.
